Training & Testing - Non Smote
Before splitting and training our dataset we need to define dependant and
independent variables. The dependant variable known as X and the independent
value known as Y. The first part is we split the dataset into two subsets, with a
portion for training. We split it by 80/20 80% for training and 20% for testing.
We trained the different models using the training dat. This involved feeding the
training data into the model, allowing the model to learn from this data.
We upsampled and downsampled for more accurate results. We upsampled the
minority class and downsampled the majority class.
We upsampled and downsampled because of the class imbalance issue. We felt
because the majority class is quite imbalanced it can lead to a bias in the model
towards the majority class and a poor performance in predicting the minority
class.
We upsampled by duplicating existing samples, we felt it would have helped the
model to learn from more examples of the minority class and improve its ability
to correctly predict it.
We downsampled by decreasing the number of samples in the majority class by
randomly removing samples. This can help to balance the class distribution and
prevent the model from being biased towards the majority class.
We felt they may have been drawbackswith this technique such as a risk of
overfitting, which is why we applied a second sampling technique
